# eBookService
This web application offers services to manage an online library.

## Back-end
Used technologies:
- Java language
- Spring framework
- Hibernate
- AWS Service (S3)
- JavaMail

## Front-end
Used technologies:
- Javascipt
- React
- CSS
- HTML





This application was created to be tested locally.




WARNING!!!Edit the properties file with your AWS and email credentials.
